# Student lifestyle behavior Analysis & Clustering

This project is about discovering student lifestyle patterns using Exploratory data analysis (EDA) along with KMeans clustering.

The goal is to find natural student behavior groups based on :
- study habits
- sleep hours
- physical activity
- stress level 
- social media usage
- CGPA

## Project workflow

- Data cleaning & preprocessing
- Exploratory data analysis (EDA)
- Feature engineering
- Feature scaling
- KMeans clustering + PCA visualization
- Cluster interpretation

## Main findings

The project finds 3 main groups : 
1. High-performance and structured students
2. High stress + low recovery group
3. Low engegement & high distraction group

The analysis also shows relationships between :
- CGPA and study hours
- Stress level + physical activity
- Depression & sleep hours

## Conclusion

The project contains diffrent features that makes it feature-rich, but the relationship between those features is often non linear because its a behavior analysis. So it makes it a real challanging project but with alot of learning value.
